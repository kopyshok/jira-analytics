# UI improvement queue

Эта папка хранит идеи по улучшению интерфейса. Формат рассчитан на работу человека и ИИ-агента.

## Правила работы

- Один экран или один цельный сценарий - один файл.
- Не удалять идеи после реализации. Перемещать файл между папками.
- Перед началом работы переместить файл из `backlog/` в `in-progress/`.
- После реализации и проверки переместить файл в `done/`.
- Если идея отклонена, переместить файл в `rejected/` и указать причину внутри файла.
- Картинки, макеты и скриншоты хранить в `assets/`.
- При перемещении файла обновлять таблицу ниже.

## Состояния

- `backlog/` - идея описана, но ещё не взята в работу.
- `in-progress/` - идея сейчас реализуется.
- `done/` - идея реализована и проверена.
- `rejected/` - идея осознанно не будет реализована.
