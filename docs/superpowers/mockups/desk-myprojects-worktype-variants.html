Плитка «Мои проекты» — разбивка по видам работ
Четыре направления для плитки «Мои проекты» на рабочем столе аналитика. Сейчас в строке проекта показаны только часы аналитика (план/факт) и одна полоса. Идея — раскрыть каждый проект на четыре вида работ: Анализ, Разработка, Тестирование, ОПЭ — плюс общий итог по проекту. Прокрутите вниз, чтобы сравнить варианты A–D на одних и тех же трёх проектах. Важно про факт. План по каждому виду работ есть всегда (он на инициативе). А вот факт по разработке, тестированию и ОПЭ — это часы всей команды, а не только аналитика, и он тяжелее считается. Поэтому варианты A и D показывают разбивку только по плану + общий факт, а B и C — полный план/факт по каждому виду.
